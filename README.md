## Usage howto
1. clone this repo and change your current directory:
 ```bash
git clone git@github.com:mb6ockatf/doodle_jump.git && cd doodle_jump
```

2. start game:
```bash
python3 src/start.py
```

`insert your nickname in main menu to save the record`
